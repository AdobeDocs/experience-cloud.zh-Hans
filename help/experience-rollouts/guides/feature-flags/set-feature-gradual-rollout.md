# 设置功能以逐步推出 {#gradual-rollout-feature}

在&#x200B;**基本详细信息**&#x200B;选项卡中配置了功能标志的百分比转出。 您可以随时在转出过程中向上或向下调整此值。

## 工作原理 {#how-it-works}

当您设置百分比转出（例如，25%）时，定义受众的该百分比将显示在该功能中。 剩余的百分比将放置到接收默认体验的&#x200B;**控制组**&#x200B;中。

您可以随着时间的推移增加或减少百分比以展开或减少转出。 将百分比降低至0%实际上会关闭适用于受众中所有人的功能，而不会删除标记。
