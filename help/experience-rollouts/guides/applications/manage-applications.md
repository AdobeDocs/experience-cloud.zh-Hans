# 管理应用程序 {#manage-applications}

Experience转出中的&#x200B;**应用程序**&#x200B;表示您要使用功能标志控制的服务或产品。 在创建功能标记之前，您必须将至少一个应用程序载入控制台。

## 谁可以管理应用程序 {#permissions}

只有具有&#x200B;**管理员**&#x200B;角色的成员才能添加或修改应用程序。 如果您没有此角色，请联系您的团队管理员。

## 添加应用程序 {#add-application}

若要向团队添加应用程序，请参阅[将应用程序载入](onboard-your-application.md)以获取分步说明。

## 团队所有权 {#team-ownership}

每个应用程序只归一个团队所有。 只有该团队的成员可以管理应用程序的功能标记。

此所有权模型还会影响特征组的结构方式：

* **在您的团队内** — 产品发布所有者可以跨同一团队所拥有的应用程序对功能标记进行分组。 请参阅[功能组以控制多个功能](../../concepts/feature-groups-to-control-multiple-features.md)。
