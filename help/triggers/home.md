# Adobe Experience Cloud Triggers{#triggers-home}

了解新的触发器UI并了解如何使用它。

您可以使用Experience Cloud触发器通过管道在Adobe Campaign和Adobe Analytics之间连接数据。 管道从您的网站中检索用户的操作或触发器。 放弃购买是触发器的一个示例。 在Adobe Campaign中处理触发器可近乎实时地发送电子邮件。
