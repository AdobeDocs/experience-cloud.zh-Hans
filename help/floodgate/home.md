# 了解Adobe Target Floodgate {#get-started}


## 什么是Floodgate？

Floodgate是一个功能管理和实验系统，可帮助应用程序安全、渐进地以更低的风险提供新的体验。

Floodgate不会为每个人同时启用新功能，而是允许产品团队：

* 逐步发布功能
从一小部分用户开始，然后随着时间的推移而增加。 如果某些功能无法按预期运行，则可以快速关闭该功能，而无需停机或重新安装软件。

* 定位特定受众的功能
为选定组（例如，测试版计划、某些区域或特定计划）启用新功能，以便在变更广泛可用之前可以验证和优化变更。

* 运行试验和A/B测试
尝试某个功能的不同版本（例如备用布局、流程或推荐）并使用数据指导产品决策，从而衡量哪个功能更有效。

* 降低关键版本的风险
大型更改可以分解为更小的、受控制的转出，因此更容易检测、理解和修复问题，以免它们影响每个人。

* 快速响应问题
如果出现错误或性能问题，团队可以仅关闭受影响的功能，使应用程序的其余部分保持稳定。

在实践中，Floodgate可帮助最终用户体验到更频繁的改进，并且中断更少：新功能可以更快地到达，更仔细地测试，并根据实际反馈快速调整。
